# 🎯 Discord Keyword Bingo Bot

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

Discordのステージチャンネルやボイスチャンネルのイベントを盛り上げる、「キーワード式ビンゴ大会」を簡単に開催できるDiscord Botです。
参加者はキーワードを選んでオリジナルのビンゴカードを発行でき、抽選からリーチ・ビンゴの申告まで、すべてDiscord上で完結します！

## ✨ 主な機能と特徴

- **🔠 オリジナルキーワードビンゴ**
  数字ではなく、あらかじめ設定した「キーワード」を使ったビンゴ大会が開催可能です。イベントのテーマに合わせた単語リストを作成できます。
- **🖼️ 画像でビンゴカードを生成**
  参加者一人ひとりに、5x5のビンゴカード画像を自動生成してDMでお届けします。マーク済みの状態も画像で視覚的にわかりやすく更新されます。
- **🎯 好きなキーワードを選択可能**
  カード発行時、参加者は好きなキーワードを4つ選んで配置することができます。
- **📣 自動判定＆通知システム**
  運営が抽選を行うと、参加者のカードが自動で判定されます。ヒットした場合や、リーチ・ビンゴ達成時にはDMへ通知が届き、専用ボタンからスムーズに会場（ステージチャンネル）へ申告できます。
- **🛡️ 安全な管理者パネル**
  管理者権限を持つユーザーのみが操作できる専用パネルから、大会の開始・抽選・リセット・終了を直感的に操作可能。権限外アクセスの防止などセキュリティを考慮した設計になっています。

## 🚀 セットアップ方法

1. **前提条件**
   Python 3.10以上がインストールされていること。

2. **依存パッケージのインストール**
   ```bash
   pip install -r requirements.txt
   ```

3. **日本語フォントの準備**
   画像生成に使用する日本語フォント（Noto Sans JP）をダウンロードします。
   ```bash
   python download_font.py
   ```

4. **環境変数（シークレット情報）の設定**
   `.env.example` をコピーして `.env` ファイルを作成し、DiscordのBot Token等を設定します。
   ⚠️ **注意**: トークン等のシークレット情報をハードコードしたり、Git等にコミットしないよう厳重に管理してください。
   ```env
   DISCORD_BOT_TOKEN=your_bot_token_here
   APP_ENV=production
   LOG_LEVEL=INFO
   ```

5. **キーワードの設定**
   `keywords.txt` に、ビンゴで使用するキーワードを1行ずつ記述します（※ゲームの性質上、最低25個以上の単語が必要です）。

6. **Botの起動**
   ```bash
   python bot.py
   ```

## 🎮 使い方

### 👑 運営（管理者）の手順
1. Discord内で `/setup` コマンドを実行し、大会に使用する各チャンネルを指定します。
   - `admin_channel`: 管理者用パネル（進行操作）を設置するチャンネル
   - `participant_channel`: 参加者用の「カード発行ボタン」を設置するチャンネル
   - `stage_channel`: 抽選結果やビンゴ達成者のアナウンスを行うチャンネル
2. 管理者パネルの **[開始]** ボタンを押して、参加受付をスタートします。
3. イベント進行中、**[抽選]** ボタンを押すと未発表のキーワードが1つ選ばれ、`stage_channel` で発表されます。
4. イベント終了後、**[終了]** ボタンを押してシステム上のデータを完全にクリア・リセットします。

### 👥 参加者の手順
1. `participant_channel` に表示される **[ビンゴカード発行]** ボタンを押します。
2. ドロップダウンから好きなキーワードを4つ選ぶと、DMに自分のビンゴカード（画像）が届きます。
3. 運営が抽選を行い、自分のカードにそのキーワードが含まれていれば、マークされた新しいカード画像がDMに自動送信されます。
4. リーチやビンゴになったら、DMに表示される **[申告する]** ボタンを押して、会場全体に知らせましょう！

## 🛠 使用技術

- **discord.py** (Discord APIラッパー)
